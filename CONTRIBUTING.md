# Contributing to Agentic TS Forecasting System

Thank you for considering contributing to this project!

## How to Contribute
1. Fork this repository
2. Clone your fork (`git clone https://github.com/YOUR_USERNAME/agentic-ts-forecasting-system`)
3. Create a new branch (`git checkout -b feature/my-feature`)
4. Make your changes
5. Add tests (`pytest`)
6. Commit (`git commit -am 'Add new feature'`)
7. Push (`git push origin feature/my-feature`)
8. Submit a Pull Request

## Development Guidelines
- Write modular, well-documented code
- Include or update relevant unit tests in `test_agents.py`
- Run tests before pushing: `pytest -v`
- Respect the modular agent structure

## Code of Conduct
Be respectful, collaborative, and constructive.
